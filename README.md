# Customer Behavior Analytics

Python | SQL | Power BI

## Project Overview
This project focuses on analyzing customer shopping behavior using Python, SQL, and Power BI. The dataset was cleaned and analyzed to understand purchasing patterns, customer segments, subscription behavior, and revenue trends. Interactive dashboards were built to convert raw data into business insights.

## Dataset
Dataset: Customer Shopping Dataset  
Records: ~3,900+ customers

Key fields:
- Customer ID
- Age
- Gender
- Category
- Purchase Amount
- Subscription Status
- Shipping Type
- Review Rating
- Season

## Tools & Technologies
- Python (Pandas, NumPy, Matplotlib)
- SQL
- Power BI
- Excel / CSV

## Project Workflow

### Data Cleaning & Preparation
- Imported dataset using Python
- Handled missing values
- Standardized column names
- Created age groups for analysis
- Prepared data for dashboard reporting

### Exploratory Data Analysis (EDA)
- Revenue trends by category
- Customer segmentation analysis
- Purchase behavior analysis
- Subscription and demographic analysis

### SQL Analysis
Performed queries for:
- Revenue by category
- Customer purchase behavior
- Subscription-based insights
- Age-group analysis

### Dashboard Features
Interactive dashboard includes:

✔ Total Customers: 3.9K  
✔ Total Revenue: 233K  
✔ Average Purchase Amount: $59.76  
✔ Average Review Rating: 3.75  

Visualizations:
- Subscription Status Analysis
- Revenue by Category
- Sales by Category
- Revenue by Age Group
- Sales by Age Group
- Interactive Filters

## Key Business Insights
- Clothing generated the highest revenue and sales.
- Middle-aged customers showed stronger purchase activity.
- Subscription behavior highlighted valuable customer segments.
- Customer demographics influenced purchasing patterns.
